# Patterns
This is a Java console application that shows how to print various patterns using nested loops.
The program enables the user to enter the number of rows (n) and prints various star (*) and number patterns depending on the chosen pattern.
It is intended to enhance the following concepts:
Nested loops
Control variables in loops
Pattern logic
Formatting console output

Technologies Used

Language: Java

Input Handling: java.util.Scanner

IDE: Any Java-supported IDE (IntelliJ, Eclipse, VS Code, etc.)
